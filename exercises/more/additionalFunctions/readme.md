## **La fonction 'len()'**
Retourne le nommbre de caractères d'une chaînne et d'une liste.
```py
len("Python")     #  6
len([1, 2, 3])    #  3
```
## **'Round()'**
```py
round(2.2)        #  2
round(2.7)        #  3
```
## **'Min()' et 'Max()'**
```py
min([1, 2, 3])    #  1
max([1, 2, 3])    #  3

min("qwertz")     #  e
max("qwertz")     #  z
```
## **'sum()'**
```py
sum([10, 10, 10]) # 30
```
## **'range()'**
Créée un interval depuis Python 3
```py
range(5)
range(2, 5) # 5 est exclu
```
